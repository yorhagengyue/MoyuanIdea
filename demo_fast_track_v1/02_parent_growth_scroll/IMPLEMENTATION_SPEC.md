# 家长成长卷轴 Demo 实施规格

## 1. 想让老板看到什么
家长端不是通知中心，而是“成长感知入口”。
打开就看懂今天学了什么、孩子变了什么、下一步是什么。

## 2. 最小页面结构（手机端）
- 今日课程卡（出勤+课程主题）
- 今日作品 Hero 卡
- Proud Moment（一句高光表达）
- 老师一句点评
- 下次预告 + 今晚轻任务入口

## 3. 数据来源
- 出勤来自 LessonSession
- 作品来自 ArtworkSnapshot
- 高光来自 SpeechHighlight / TeacherNote
- 下次预告由系统模板生成并可老师轻改

## 4. 演示时最好用的真实例子
家长下班后打开页面看到：
“今天主题：山的不同叫法。孩子第一次说出完整原因句：我觉得这是峰，因为它尖尖的。下次会练‘岭和崖’的区别。”

## 5. 为什么这块特别容易打动人
因为家长痛点是“看不见学习过程”，不是“收不到通知”。
这个页面直接解决焦虑，老板听完会立刻理解商业价值。

## 6. 你们要完成的具体产物
- 一个手机端卷轴详情页（当天）
- 一个最近 3 天列表页
- 一条自动组装逻辑（把四类数据拼成一页）

## 7. 验收
- 家长在 30 秒内说得出“今天学了什么+哪里有变化+下次做什么”
- 每节课都能稳定产出一页卷轴，不靠手工复制粘贴
