# Demo 总演示脚本（7分钟）

## 0:00 - 0:40 开场
先讲一句：我们不是做功能堆叠，而是把“课堂 -> 课后 -> 持续学习”连成一条线。

## 0:40 - 2:40 Demo 1：课堂受控 AI
老师一键开课，孩子点选或说一句，系统只围绕当前知识点回应。
如果噪音大或识别低，自动切 fallback 按钮，不中断课堂。

你要强调：AI 在课堂里是“受控协作”，不是自由聊天。

## 2:40 - 4:20 Demo 2：家长成长卷轴
下课后，家长手机端直接看到四件事：
今天学了什么、孩子作品是什么、一句高光表达、下次接什么。

你要强调：家长不再翻群消息，30秒看懂孩子变化。

## 4:20 - 6:20 Demo 3：孩子共创角色
孩子自己创建角色，做一次 5 分钟任务对话。
结束后自动生成角色进阶卡：完成目标、升级句子、新词战利品、下一关挑战。

你要强调：不是一次性练习，孩子有动力回来继续学。

## 6:20 - 7:00 收口
最后一句固定：
我们把“课堂可用、家长可见、孩子可持续”一次性做成闭环，这就是核心竞争力。
