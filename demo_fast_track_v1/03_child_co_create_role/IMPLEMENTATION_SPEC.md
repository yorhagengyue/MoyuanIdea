# 孩子共创角色 Demo 实施规格

## 1. 想让老板看到什么
孩子不是被动练习，而是“我创造角色，我带着角色升级”。
这是新颖感和复访动力的来源。

## 2. 最小流程
创建角色（3-5分钟）-> 一次任务对话（5分钟）-> 角色进阶卡（自动生成）。

## 3. 角色创建字段
名字、身份、性格、口头禅、梦想目标、视觉元素。
创建后先系统初审，再老师确认，确认后进入班级可见。

## 4. 固定技能树（本轮不开放孩子修改）
口语表达、句型完整度、词汇使用、语法准确度。

## 5. 对话策略
只用“提问引导”，不做直接否定。
例如孩子说“我画山”，系统问“你能再加上颜色和位置吗？”

## 6. 进阶卡必须有四块
- 今天完成了什么
- 把哪句升级成了更完整句
- 今天新词战利品
- 下一关 3 分钟挑战

## 7. 演示时最好用的真实例子
同一个任务，不同角色会有明显不同语气。
角色 A 偏鼓励：先肯定再追问。
角色 B 偏侦探：用线索式提问引导完整表达。
老板会一眼看出不是模板复制。

## 8. 验收
- 孩子 5 分钟内能创建并开跑第一轮任务
- 每次任务结束都能自动生成进阶卡
- 连续 3 次任务能看出成长轨迹，不是重复文案
