# 第三部分测试与验收清单（汉字声像行动课）

更新时间：2026-03-25

## 一、功能场景

1. 正常课堂  
- 每个孩子设备可完成至少 6 字流程。  
- 互动题正常提交并返回即时反馈。

2. 弱网场景  
- 动画加载失败时自动降级到静帧 + 旁白。  
- 课堂流程不中断。

3. 连续误答场景  
- 孩子连续两次答错后自动降难。  
- 不出现长时间卡题。

4. 易混字场景  
- 系统可记录混淆对（如 `人/入`、`犬/大` 的近形误判场景扩展位）。  
- 反馈中给出简短纠偏提示。

## 二、演示验收（主线）

- 8 分钟内跑通“字形变换 -> 孩子互动 -> 结果面板”。  
- 必须出现 `犬->犭` 形变高记忆点。  
- 非技术观众可在演示后说清“为什么孩子更容易记住字”。

## 三、教学验收（次线）

- 单节课平均互动次数 >= 8。  
- 课末即时识别正确率较课前提升 >= 20%。  
- 报告可直接显示：已掌握字、易混字、下节建议。

## 四、回归重点

- AI 回复是否严格在字卡知识范围内。  
- 老师一键重播/下一字是否可用。  
- 结果面板三项指标是否完整：识别速度、记忆保持率、易混字错误率。
