# 动画生成提示词库（第三部分）

更新时间：2026-03-25

## 统一提示词框架

用于生成单字动画时，固定使用以下结构，避免风格漂移：

1. 字目标：`{target_char}`  
2. 年级段：`P1-P3`  
3. 时长：`30-60秒`  
4. 视觉风格：`儿童友好、线条清晰、背景简洁`  
5. 叙事结构：`图像来源 -> 形体变化 -> 现代字形 -> 一个词语应用`  
6. 认知目标：`只强调一个知识点`

## 通用系统提示（给生成引擎）

你是低龄汉字动画脚本助手。  
只输出儿童可理解的短句中文。  
每条旁白不超过16字。  
每个字只讲一个核心记忆点。  
禁止引入与目标字无关的历史细节。  
若字源争议较大，采用“教学可理解版本”并标记“课堂解释用”。  

## 示例提示词：犬 -> 犭

请生成一个45秒中文微动画脚本，目标字为“犬”，并展示“犬在偏旁中常写作犭”。  
受众是 P1-P3。  
要求输出4个镜头：  
1) 小狗形象出现  
2) 线条简化成“犬”  
3) “犬”缩写到左侧变成“犭”  
4) 用“狗、猫、狼”做偏旁例子  
旁白每句不超过16字，语气亲切，不说术语。

## 示例提示词：人 -> 亻

请生成40秒微动画脚本，讲“人”到“亻”的变化。  
强调点：在左边当偏旁时写作“亻”。  
结尾给一个互动问题：“你能说出一个带亻的字吗？”  

## 安全与可用边界

- 不生成可能误导的复杂古文字考据。  
- 不使用成人化表达。  
- 不输出超长旁白。  
- 优先让孩子“看懂并记住”，而不是讲深学术背景。
